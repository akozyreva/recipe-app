## Recipe app
Little recipe app with Django Rest Framework and Nuxt.js
### Requirements
Python 3.x
Nuxt.js 2.4.0
### Istallation
#### Backend installation
With pipenv.
```
cd api
# install requirements for backend
pipenv install
pipenv shell
python manage.py runserver
```
#### Frontend installation
```
cd client
npm install
npm run dev
```
